Step by Step Tutorial for Ambari Installation

VM Deployment

  1. Login VMware vsphere client system
  2. Create 3 VMs with specific system requirements
    1. Set the name of VM
    2. Choose data storage
    3. Choose hardware configuration
    4. Load centos6 ios file into CD-ROM when booting
      You can use minimum version of centos. Then you need to click the option Running system installation with system booting.
    5. Configuration complete

  3. Operating system installation

    1. Power on the VM

    2. Choose the first one.

    3. Click skip to continue.

    4. Choose language, keyborad and timezone.


    5. Set root password.

    6. Click Using entire drive to continue.

    7. Choose Write changes to disk to continue.

    8. Reboot.

VM Preparation

1. Configuration and Installation for VMs(master and slave servers)

  1. Install java.

  2. Enable network adapter.

    Then, enable it to run after system booting

  3. Using tool ping to check status of accessing public network and local network.

  4. Change systems hostname on all servers.

    1. Update file(/etc/sysctl.conf), add a item

    2. Update file(/etc/hosts), add doamins.

    3. Update file(/etc/sysconfig/network)

    4. Reboot systems

    5. Check the correction of hostname

  5. Set up password-less SSH on all servers

    1. Disable SELinux

      Set disabled for SELinux:

    2. Enable access for using public key.
      Warning: File name is sshd_config, not ssh_config.

      Delete comment signal(#) blow:

    3. Generate public key and private key on master server.

      Then, click default option for configuration.

    4. Create directories and copy public key to slave servers.

      Login to master server, then copy public key:

      Login to slave server, then copy content from public key to a file(authorized_keys):

  6. Install ntpd on all servers.

    1. Check to see if you have installed ntpd.

    2. If not, install ntpd

  7. Close transparent huge page on all servers.

    1. Append code block to file(/etc/rc.local) for closing transparent huge page.

      code block:

    2. Reboot operating system.

    3. Check the status of transparent huge page to ensure the status is never.

2. Set up a local repository on master server.

  1. Install local repository tool.

  2. Install http service.

  3. Start http service.

  4. Install tool wget.

  5. Download ambari & hdp package files to master server.

  6. Copy ambari and hdp packages to http dir, then untar them.

  7. Check repository address for next step.

    HDP base URL and gpgkey:

    HDP-UTILS base URL and gpgkey(gpgkey file same with HDP gpgkey file):

  8. Copy repository config files.
    ambari repo file:

    hdp repo file:

  9. Update repository config files.
    You ought to renew base URL and gpgkey URL in your ambari.repo and hdp.repo files.

3. Ambari-server setup on master server

  1. Install ambari server

  2. Run command for ambari setup.

  3. If you are not disable the SELinux, you need to press y.

  4. If your ambari server under user root, you just need to press n.

  5. If you are not disable iptalbes, you need to press y to continue.

  6. Select JDK version, then follow steps to install it.

  7. Choose database type
    You can choose embeded database. If you choose mysql, you need to install mysql database on master server and input info for ambari setup. Then put a mysql connector into specific path and run a command below to specify the path of connector.

  8. Run ambari-server

4. Ambari-agent setup on slave servers.

  1. Copy ambari-agent rpm file to slave servers.

  2. Install ambari-agent.

  3. Renew ambari-server URL in config file.

    Renew ambari-server URL below:

  4. Run ambari-agent.

Installation

Before you launch the ambari installation webUI, you ought to ensure things below.

  1. Start the service httpd.
  2. Stop the service iptables.
  3. Start the ambari-server on master server and ambari-agent on slave servers.
    If you want to start quickly, it is advisable to choose default options.
  4. Open your brower, then access to the WebUI and click the item (Launch Install Wizard)
    URL: 192.168.110.150:8080
    Account: admin
    Password: admin

  5. Get Started
  6. Select Version
    Choose local repository, then input base URL of HDP and HDP-UTILS.
  7. Install Options
    Input your masters hostname and slaves hostnames. Then copy your masters ssh private key into item.

  8. Confirm Hosts

    Just click ok to continue.

  9. Choose Services
    Choose what service you want to add into your system. You can add new services into your system later on.

    Just click ok to continue.

  10. Assign Masters
  11. Assign Slaves and Clients
  12. Customize Services
    You just need to input info for items with red exclamation mark.
  13. Review
  14. Install, Start and Test
    If you get to this step successfully, you can drink a cup of coffee and enjoy this time.
  15. Summary
    So now, say hello world to Ambari.
